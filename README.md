# conversando-com-ia-usando-whisper
Projeto de assistente virtual por voz que utiliza IA para converter fala em texto, processar com ChatGPT e responder em áudio, integrando tecnologias como Whisper e Google Text-to-Speech.

---


### 📌 Assistente de Voz com IA (Whisper + ChatGPT + gTTS)

---

## 📖 Sobre o Projeto

Este projeto foi desenvolvido como desafio prático da plataforma Digital Innovation One (DIO), com o objetivo de aplicar na prática conceitos de Inteligência Artificial e processamento de voz.

A aplicação permite interação por voz com um assistente inteligente, utilizando tecnologias modernas de IA para transformar fala em texto e gerar respostas em áudio.

---

## 🧠 Funcionalidades

* 🎤 Captura de áudio do usuário
* 📝 Conversão de fala para texto (Speech-to-Text)
* 🤖 Processamento com IA (ChatGPT)
* 🔊 Conversão de texto para fala (Text-to-Speech)
* 🌎 Suporte a múltiplos idiomas

---

## ⚙️ Tecnologias Utilizadas

* Python
* Whisper
* ChatGPT
* Google Text-to-Speech
* Google Colab

---

## 🔄 Fluxo da Aplicação

1. O usuário fala no microfone
2. O áudio é convertido em texto usando o Whisper
3. O texto é enviado para o ChatGPT
4. A IA gera uma resposta inteligente
5. A resposta é convertida em áudio com gTTS
6. O áudio é reproduzido para o usuário

---

## 🎯 Objetivo

Explorar a integração entre diferentes APIs de Inteligência Artificial para criar uma experiência interativa baseada em voz, simulando um assistente virtual.

---

## 💡 Diferenciais

* Integração completa entre voz + IA
* Comunicação natural com o usuário
* Aplicação prática de IA em tempo real
